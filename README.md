# MobileProject

This repo contains two parts:

- `gift-box/` - Laravel backend (API, admin, payment integration).
- `giftbox/` - Android app (Gradle project).

## Layout

- Backend entry point: `gift-box/public/index.php`
- Android app module: `giftbox/app/`

## Notes

- Android build artifacts and Gradle caches are ignored via the top-level `.gitignore`.
